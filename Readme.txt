Ionic 2 Web Application

Steps to run the Web Application:

1. install ionic and cordova using command prompt :
	npm install -g cordova ionic
				(install nodejs if not installed)
2. download and create ionic v2 project :
	 ionic start apiTechScan blank

3. Replace the src folder with the folder(src) given in deepRepo.

4. go to the apiTechScan folder created
	cd apiTechScan

5. Run the Web Application using command prompt.
	ionic serve
